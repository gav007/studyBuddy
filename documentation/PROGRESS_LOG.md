# StudyBuddy Current Status

- Current focus: exploring whether AI can support beginner coding through a simple StudyBuddy project.
- The project is a small Python app that reads study session data from a CSV file.
- `main.py` asks the user for a file path.
- `modules/file_handler.py` has helper functions for reading the CSV header and returning the data rows.
- `modules/display_table.py` displays the CSV data neatly in the console.
- The current goal is to understand data flow clearly:
  CSV file -> header -> rows -> display table.
- The project is still at prototype stage.
- The priority is learning Python step-by-step, not generating a finished app too quickly.
